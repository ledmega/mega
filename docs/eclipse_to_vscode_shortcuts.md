# 이클립스(Eclipse) vs VS Code / Cursor 단축키 비교 가이드

이클립스에 익숙하신 개발자분들이 VS Code나 Cursor IDE에 빠르게 적응하실 수 있도록 자주 사용하는 단축키를 비교 정리했습니다.

*(기준: Windows/Linux 기본 단축키 설정)*

## 🔎 파일/코드 검색 및 탐색

| 기능 설명 | Eclipse | VS Code / Cursor | 비고 및 활용 팁 |
| :--- | :--- | :--- | :--- |
| **빠른 파일 열기** (파일 이름으로 찾기) | `Ctrl` + `Shift` + `R` | **`Ctrl` + `P`** | 파일명의 일부만 입력해도 퍼지(Fuzzy) 검색 지원 |
| **전역 텍스트 검색** (모든 파일에서 찾기) | `Ctrl` + `H` | **`Ctrl` + `Shift` + `F`** | 정규표현식, 대소문자 구분, 특정 파일 패턴 포함/제외 가능 |
| **클래스/메서드 이름으로 찾기** | `Ctrl` + `Shift` + `T` | **`Ctrl` + `T`** | 워크스페이스 내의 모든 심볼(클래스, 함수 등) 검색 |
| **파일 내 특정 줄(Line)로 이동** | `Ctrl` + `L` | **`Ctrl` + `G`** | |
| **선언된 곳으로 이동** (메서드/변수 추적) | `F3` 또는 `Ctrl` + 클릭 | **`F12`** 또는 `Ctrl` + 클릭 | 구현체가 1개라면 해당 파일로 즉시 점프 |
| **구현체 찾기** (인터페이스를 상속받은 클래스) | `Ctrl` + `T` | **`Ctrl` + `F12`** | 인터페이스 메서드에서 실제 구현 코드를 찾을 때 유용 |
| **사용된 곳 모두 찾기** (Reference 검색) | `Ctrl` + `Shift` + `G` | **`Shift` + `F12`** | 팝업 창이 열리면서 해당 메서드/변수가 호출된 곳 목록 표시 |
| **이전 위치로 돌아가기** | `Alt` + `←` | **`Alt` + `←`** | (마우스 뒤로가기 버튼과 동일) 코드를 추적하다가 원래 보던 곳으로 복귀 |
| **다음 위치로 앞으로 가기** | `Alt` + `→` | **`Alt` + `→`** | 처음 위치로 다시 가고 싶을 때 (마우스 앞으로가기 버튼) |

## 💻 코드 편집 및 리팩토링

| 기능 설명 | Eclipse | VS Code / Cursor | 비고 및 활용 팁 |
| :--- | :--- | :--- | :--- |
| **자동 완성 제안** | `Ctrl` + `Space` | **`Ctrl` + `Space`** | |
| **빠른 수정 / 에러 픽스** | `Ctrl` + `1` | **`Ctrl` + `.` (마침표)** | 에러가 난 코드 위에서 누르면 해결책이나 자동 Import 제시 |
| **이름 일괄 변경** (Rename Refactoring) | `Alt` + `Shift` + `R` | **`F2`** | 변수나 함수 이름을 일광 변경 (연결된 모든 파일이 자동 수정됨) |
| **코드 자동 정렬** (Formatting) | `Ctrl` + `Shift` + `F` | **`Shift` + `Alt` + `F`** | |
| **한 줄 지우기** | `Ctrl` + `D` | **`Ctrl` + `Shift` + `K`** | |
| **줄 위/아래로 이동** | `Alt` + `↑` / `↓` | **`Alt` + `↑` / `↓`** | 해당 줄(또는 선택 블록) 전체를 이동 |
| **선택한 줄 복사하여 아래/위로 붙여넣기** | `Ctrl` + `Alt` + `↓` | **`Shift` + `Alt` + `↓` / `↑`** | |
| **파일 내 텍스트 찾기/바꾸기** | 찾기: `Ctrl` + `F`<br>바꾸기: `Ctrl` + `F` | 찾기: **`Ctrl` + `F`**<br>바꾸기: **`Ctrl` + `H`** | |

## 🛠 창 및 뷰 관리

| 기능 설명 | Eclipse | VS Code / Cursor | 비고 및 활용 팁 |
| :--- | :--- | :--- | :--- |
| **현재 창 닫기** | `Ctrl` + `W` | **`Ctrl` + `W`** | |
| **파일 탐색기 열기/포커스** | (따로 없음) | **`Ctrl` + `Shift` + `E`** | 좌측 폴더 구조 트리로 바로 이동 |
| **하단 터미널/패널 토글** | (따로 없음) | **`Ctrl` + `\`` (백틱)** | 하단에 콘솔 창이나 터미널을 열고 닫을 때 |
| **분할 창 넘나들기** | `Ctrl` + `F6` 등 | **`Ctrl` + `1`, `2`, `3`** | 에디터 창을 반으로 갈라서 볼 때, 창 번호로 포커스 이동 |

## 💡 Cursor 전용 특화 단축키 (AI 기능)
Cursor 에디터는 일반 VS Code 기능에 강력한 AI 기능이 더해져 있습니다.

| 기능 설명 | Cursor 단축키 | 활용 예시 |
| :--- | :--- | :--- |
| **코드 인라인 생성/수정 (Cmd+K 봇)** | **`Ctrl` + `K`** | 에디터에 코드를 띄워둔 채로 빈 공간에 누르면 "이 함수를 비동기로 바꿔줘" 같은 코드 작성/수정 명령 가능 |
| **AI 챗창 열기 (채팅 패널)** | **`Ctrl` + `L`** | 우측(혹은 좌측)에 뜨는 AI와의 대화방 토글. 프로젝트 전체 맥락 질문 시 사용 (`@`를 쳐서 특정 파일/폴더 멘션 가능) |
| **다음 탭 완성 (Copilot 형태)** | **`Tab`** | 희미한 색으로 제안되는 코드를 채택 |
