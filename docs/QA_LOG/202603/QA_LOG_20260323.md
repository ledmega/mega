## ❔ Q&A 및 기술 세부 사항

**Q: SSL 인증서는 어디서 발급받았나요?**  
**A:** cloudflare.com에서 무료로 발급받아 적용하였습니다. Cloudflare의 Proxy(CDN) 기능을 활용하면 간편하게 HTTPS를 적용할 수 있습니다.

**Q: 에이전트에서 `http` 대신 `https`를 사용할 때 주의할 점은?**  
**A:** Java 8 에이전트 환경에서 최신 TLS 버전을 지원하는지 확인이 필요할 수 있으나, Cloudflare의 호환성 설정을 통해 대부분 문제없이 연결됩니다. `webserver.url` 설정 시 반드시 `https://` 프로토콜을 명시해야 합니다.
