🧪 문의 시뮬레이터
실제 카카오톡/이메일 없이 봇 처리 흐름 전체를 테스트합니다.
FAQ에 등록된 키워드가 포함되면 자동 답변, 없으면 AI 초안이 생성됩니다.
⚙️ 처리 흐름
1
문의 유입 & RAW 저장
원본 데이터를 cs_inbound_data 테이블에 즉시 저장 (감사 추적)
2
상담 세션 생성/재사용
발신자 ID 기준으로 열린 세션이 있으면 이어서, 없으면 신규 세션 생성
3
FAQ 키워드 매칭
질문 내용을 FAQ 테이블과 비교. 매칭 시 즉시 자동 답변 (AUTO_REPLIED)
4
OpenAI RAG 초안 생성
FAQ 미매칭 시 FAQ 데이터를 컨텍스트로 주입하여 AI가 답변 초안 작성 (DRAFT_CREATED)
5
SSE 알림
모든 관리자 화면에 CS_EVENT로 실시간 푸시 알림 전송
💡 테스트용 샘플 질문
FAQ를 먼저 등록하면 샘플이 나타납니다.
